== Description ==
Express Checkout is a WooCommerce plugin that instantly converts the default, bloated and messy WooCommerce checkout into a beautiful, high converting checkout experience for your customers.

Optimized checkout
Fewer fields results in faster checkout.

Mobile ready
Best checkout, wherever they are.

All payments
Compatible with all payment methods.

Easy customization
Works with your brand; fonts, colors, logo.
